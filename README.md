# 🌍 Polyglot

**Real-time speech-to-speech translator built with React Native & Expo.**

---
## 📖 The Story

Polyglot was born from a real personal need. With my parents living in Portugal, I often found myself frustrated by the language barrier during daily life—whether it was hanging out with locals or following a sermon at church.

I built this app to bridge that gap: just place your phone down, start listening, and get a fluid, instant translation so you never lose the thread of a conversation or a speech again.

## 🚀 Features

* **Precision Speech Recognition**: Powered by `expo-speech-recognition` for high-accuracy voice capture.
* **Dual Mode**:
* *Interim Mode (Flash)*: Real-time translation as you speak.
* *Final Mode*: Structured sentence-by-sentence translation with a full history.


## 🛠 Tech Stack

* **Framework**: [React Native](https://reactnative.dev/) (Expo)
* **Language**: TypeScript
* **Styling**: [Uniwind](https://uniwind.dev/) (Tailwind CSS)
* **UI Components**: React Native Paper
* **Translation**: Google Translate Engine


## 📦 Installation

1. **Clone the repo**:
```bash
git clone https://github.com/your-username/polyglot.git

```

2. **Install dependencies**:
```bash
npm install

```


3. **Start the project**:
```bash
npx expo start

```
